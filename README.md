# 장산이 올인원 💰

개인 재무 관리 + 여행 플래너 올인원 앱. Google 계정으로 로그인하면 모든 기기에서 데이터가 실시간 동기화됩니다.

---

## 기능 목록

### 🔐 로그인 / 동기화
- Google 계정으로 로그인 (팝업 → 리다이렉트 자동 폴백)
- Firestore 실시간 저장 (디바운스 1.5초)
- 모든 기기에서 데이터 자동 동기화

---

### 🏠 대시보드
- 이번 달 잔여 예산 배너 (수입 − 지출)
- 저축률 및 저축금액 표시
- 수입 / 지출 / 잔액 / 건수 요약 카드
- 변동 지출 도넛 차트 (이번 달 / 3개월 평균 전환)
- 총 자산 및 주식 평가액 현황
- 최근 가계부 내역 빠른 확인
- 월 이동 (이전/다음 달)
- 한 달 요약 보고서 다운로드 (Excel)

---

### 💰 수입 / 지출
- 수입 항목 추가 / 수정 / 삭제
- 고정 지출 항목 추가 / 수정 / 삭제
- 변동 지출 항목 추가 / 수정 / 삭제
- 예산 카테고리 관리 및 실지출 연동
- 예산 자동 설정 (✨ AI 제안)
- 매달 반복 항목 기본값 설정 (드래그로 순서 변경 가능)
- 자금 분배 계산기 (총 자산 연동 지원)

---

### 💳 신용카드
- 카드별 월 납부 대금 기록
- 할부 개월 수 / 결제일 관리
- 이번 달 전체 결제 완료 일괄 처리
- 할부 계산기 (원금, 이자, 월 납부액, 총 상환금액)
- 카드별 연이율 설정

---

### 🏦 자산 목록
- 자산 항목 추가 / 수정 / 삭제 (분류별 관리)
- 총 자산 합계 표시
- 주식 포트폴리오 관리
  - 국내주식 (종목 코드 기반 현재가 자동 조회)
  - 해외주식 / 금현물 (직접 입력)
  - 매입금액 대비 평가손익 표시

---

### 🗓️ 캘린더
- 연간 저축 캘린더 (연도 이동)
- 월별 소비 일정 및 저축 목표 기록
- 식비 캘린더 연동 (예산현황 자동 반영)
- 식비 직접 설정 모드
- 소비 금액 보기/숨기기 토글

---

### 📒 가계부
- 빠른 입력 (유형 / 날짜 / 카테고리 / 메모 / 금액)
- #태그 입력 및 태그 자동완성 드롭다운
- 키워드 자동 규칙 (메모 입력 시 카테고리 자동 적용)
- 자동화 항목 (매월 지정일에 자동 등록)
- 카테고리 관리 (아이콘 포함)
- 내역 검색 (날짜 / 카테고리 / 메모 / 금액 범위)
- 유형 / 카테고리 / 태그 필터
- Excel 내보내기
- 월 마감 (소감 기록 + 아카이브 저장)

---

### 📊 분석
- 월별 수입/지출 추이 차트
- 카테고리별 지출 비중 분석
- 지출 성격별 분류 (고정/변동/투자 등 커스텀)
- 저축률 추이

---

### 🗃️ 마감 아카이브
- 마감된 월별 기록 조회
- 월별 소감 확인
- 과거 데이터 검색 및 복기

---

### ✈️ 여행 플래너 — 내 여행
- 여행 추가 / 수정 / 삭제 (이름, 구분, 날짜, 여행지, 동행자, **링크**)
- 예정 / 진행 중 / 완료 상태 자동 구분
- 카드에 링크 뱃지 표시 (새 탭 열기)
- 상세 페이지 내 링크 바로 열기
- 여행 일정 관리 (일정 목록 / 시간표 / 요약 뷰)
- 예약 목록 (항공 / 숙소 / 기타)
- 지출 기록 (외화 자동 환산, 환율 직접 설정)
- 가고싶은 곳 / 버킷리스트 관리
- 연간 캘린더 자동 연동 (여행 일정 표시)
- 준비물 체크리스트 (공통 / 해외 전용)
- 여행 팁 게시글 작성 및 태그 관리

### ⭐ 여행 플래너 — 버킷플레이스
- 가고 싶은 여행지 목록 관리
- 카테고리 / 태그 필터
- 방문 완료 체크

---

### ⚙️ 설정
- 앱 전반 설정 관리
- 데이터 월별 삭제 (삭제 제외 항목 지정 가능)
- 지출 성격 설정

---

## 파일 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 앱 진입점, Firebase 인증 인라인 포함 |
| `style.css` | 전체 스타일 |
| `app.js` | 핵심 앱 로직 (대시보드, 가계부, 자산, 캘린더 등) |
| `travel.js` | 여행 플래너 모듈 |

> `firebase-sync.js`는 `index.html`에 통합되어 별도 파일 불필요.
